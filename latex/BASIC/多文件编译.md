# cmd
```tex
\input{file}	% 放在导言区，用于导入包含导言的文件；通常不要把\newcommand{}放里面，容易重定义
\include{file}	% 放在正文，用于导入包含正文的文件，另起一页
\include*{file}	% 放在正文，用于导入包含正文的文件，不另起一页 ; 需要newclude包
\includeonly{filelist}	% 放在导言区域，用于规定需要编译的包含正文的文件
```
# example 
## 先写主文件(main.tex)
```tex
\input{begin}

\includeonly{qianyan,end}
\begin{document}
	\include*{qianyan}
	\include*{end}
	\include{end}
\end{document}
```
